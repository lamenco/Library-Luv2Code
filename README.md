# Library-Luv2Code
Project Description
This project is a web application for an online book store where users can browse, search for, and order books. In addition, users can rate and comment on the books.

Project Features
Registration and Login: Users can create an account and log in using OAuth with JWT (JSON Web Tokens) for authentication.
Book Catalog: Users can browse a list of available books, view details for each book, and search for books based on various criteria.
Adding Books to Cart: Users can add books to their shopping cart and proceed with the checkout process.
Rating and Commenting: Users can rate books on a scale from 0 to 5 and leave comments on the books.
User Management: Administrators have the authority to manage users, assign roles, and block or delete user accounts.
Technologies and Tools

Backend (Spring Security):
Java
Spring Boot for creating RESTful APIs
Spring Security for managing authentication and authorization
JWT for token generation and management
Database (e.g., MySQL or PostgreSQL) for data storage

Frontend (React):
JavaScript and React for creating the user interface
Axios or Fetch for making requests to the Backend API
Redux or Context API for state management
HTML and CSS for design and styling
